    EditAvailabilityDialog

    Class representing the "Edit Availability" dialog that enables users to set when they are available for tutoring appointments.

    Constructor

    new EditAvailabilityDialog(textFieldEl, profile)

    Creates a new "Edit Availability" dialog that updates the given text field's value and update the profile (if given).
