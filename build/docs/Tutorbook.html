    Tutorbook

    Class that represents uppermost level of our web app.

    Constructor

    new Tutorbook()

    Creates a new Tutorbook object:

    1. Initializes the website's configuration data.
    2. Signs in the user with Firebase Authentication.
    3. Initializes all app views and packages and routes the user to their desired destination within the app.
