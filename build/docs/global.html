    app

    The window's Tutorbook web app instance.

    You can access any variables or objects stored in that web app class from anywhere in your code (e.g. window.app.render points to a Render object).

    WebsiteConfig

    A website configuration that denotes who can access the website, what locations are shown on the website, what grades can be selected on the website, etc.

    Properties:
    Name Type Description
    created external:Timestamp

    When the website was created.
    updated external:Timestamp

    The last time the website was updated.
    domains Array.<string>

    The email domains that can access this website (i.e. a user be logged in with an email that ends in one of these domains to be able to access this website configuration's app partition).
    grades Array.<string>

    The grades that are shown (and thus can be selected) on this website (e.g. ['Freshman', 'Sophomore']).
    locations Array.<string>

    The IDs of the locations shown on this website.
    url string

    The URL of the website's app partition (e.g. 'https://gunn.tutorbook.app' or 'https://woodside.tutorbook.app').
    name string

    The name of the website configuration (used when showing the user error messages about invalid login attempts).
