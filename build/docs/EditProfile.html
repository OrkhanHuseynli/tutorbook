    (async) saveImage(file)

    Saves an image to be the profile's new photo by:

    1. Changing the profile image to a loading icon that will get updated with the shared image.
    2. Uploading the image to Cloud Storage.
    3. Generating a public URL for the uploaded file.
    4. Updating the chat message placeholder with the image's URL and re-renders the profile view with the updated profile image.
